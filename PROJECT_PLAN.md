# Solana Brain Project Plan

## Goal
Build a Solana trading intelligence bot that learns for 2 months before any real trading.

## Mode
- Learning mode only
- No real buying
- No real selling
- User makes final decision

## Main Features
- Track new Solana tokens
- Store price, liquidity, volume, market cap
- Track outcomes after 5m, 15m, 1h, 4h, 24h
- Track smart wallets
- Collect trading recommendations
- Score recommendation sources
- Send Telegram daily reports
- Send Telegram alerts for strong opportunities

## Recommendation Sources
- Telegram channels
- X accounts
- Reddit
- YouTube
- Trading websites

## Reports
- Weekly learning report
- Monthly learning report
- Daily learning report: what the bot learned today
- Daily market report
- Best tokens of the day
- Worst tokens of the day
- Best wallets
- Best recommendation sources
- Failed recommendation sources

## Final Output
The bot gives a confidence score, and I decide manually.
